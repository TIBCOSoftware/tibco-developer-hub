# TIBCO BW6 Prompt Library

A curated collection of reusable prompts, implementation templates, and integration examples for **TIBCO ActiveMatrix BusinessWorks 6 (BW6)**.

This repository is designed to help developers, architects, and integration teams accelerate BW6 application development using structured prompt-driven guidance.

---

# Overview

The TIBCO BW6 Prompt Library provides categorized prompt examples covering common enterprise integration scenarios including:

- REST APIs
- SOAP Services
- JMS Messaging
- JDBC Database Operations
- File Processing
- Subprocess Orchestration
- Shared Resources
- Module Properties
- Logging and Validation Patterns

The prompts are implementation-focused and aligned with BW6 development practices.
